# JARVIS

JARVIS, Python tabanlı gelişmiş ve modüler bir yapay zeka asistan projesidir.
Proje; sesli komut sistemi, TCP istemci/sunucu altyapısı, akıllı otomasyon modülleri, yapay zeka entegrasyonları ve genişletilebilir servis mimarisi ile modern bir dijital asistan altyapısı oluşturmayı hedeflemektedir.

## Özellikler

* Sesli asistan sistemi
* Modüler mimari
* TCP Client/Server iletişimi
* Yapay zeka entegrasyonları
* Komut sistemi
* Akıllı cihaz kontrol altyapısı
* Gelişmiş log sistemi
* Çoklu servis desteği
* Ölçeklenebilir klasör yapısı
* Güvenlik ve yetkilendirme altyapısı

## Teknolojiler

* Python
* Socket Programming
* Async/Await
* AI API Entegrasyonları
* JSON Config Yönetimi
* Git & GitHub

## Hedef

JARVIS, sadece basit bir sesli asistan değil; geliştirilebilir, yüksek performanslı ve gerçek dünya senaryolarına uygun profesyonel bir AI Assistant platformu olmayı amaçlamaktadır.

## Yol Haritası

* Yerel AI model desteği
* Mobil kontrol paneli
* Web dashboard
* Uzaktan cihaz yönetimi
